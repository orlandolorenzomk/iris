== Platform Overview

Iris is a lightweight, self-hosted cloud platform built directly on Proxmox.
It creates infrastructure only on demand — using near-zero resources when idle.

== Architecture Documents

[cols="1,2", options="header"]
|===
| Document | Description

| link:nexus-architecture.html[Iris Nexus]
| Control plane API. Spring Boot 3 Modulith + GraalVM. Manages auth, spaces, deployments, and machine scheduling.

| link:prism-architecture.html[Iris Prism]
| Infrastructure daemon (Zig). Runs on each Proxmox node. Provisions VMs, LXCs, and virtual networks.

| link:aurora-architecture.html[Iris Aurora]
| Docker image registry (Python). OCI-compliant. Handles docker push/pull and image lifecycle.

| link:network-isolation.html[Network Isolation]
| How User Space network isolation is implemented using Proxmox SDN and VXLAN.
|===
